Git Command Reference

Getting Started

Initialize a repository or clone an existing one.

git init
git clone https://github.com/user/repo.git
    •
    Basic Workflow

    The essential daily Git workflow: stage, commit, push.

    git add .
git commit -m "Your message"
git push
    •
    Tip: Use git add -p to interactively stage hunks of changes.
    Viewing Changes

    Inspect the state of your repository and view differences.

    git diff
git log --oneline
    •
    Branching

    Create and manage branches for parallel development.

    git branch feature
git checkout feature
git checkout -b feature
    •
    Merging

    Combine branches together.

    git checkout main
git merge feature
    •
    Tip: Use --no-ff to always create a merge commit, even for fast-forwards.
    Rebasing

    Reapply commits on top of another base.

    git rebase main
git rebase -i HEAD~3
    •
    Warning: Never rebase commits that have been pushed to a shared branch!
    Remote Repositories

    Work with remote repositories like GitHub, GitLab.

    git remote add origin https://github.com/user/repo.git
git fetch origin
    •
    Stashing

    Temporarily save changes without committing.

    git stash
git stash pop
    •
    Undoing Changes

    Undo commits, unstage files, and restore changes.

    git reset HEAD~1
git checkout -- file.txt
    •
    Warning: git reset --hard permanently discards changes!
    Tags

    Mark specific points in history as important (releases).

    git tag v1.0.0
git push origin v1.0.0
    •
    Cherry-Pick

    Apply specific commits from another branch.

    git cherry-pick abc1234
    •
    Git Bisect

    Binary search to find the commit that introduced a bug.

    git bisect start
git bisect bad
git bisect good v1.0.0
    •
    Git Reflog

    View history of HEAD changes to recover lost commits.

    git reflog
git checkout HEAD@{2}
    •
    Lifesaver: Reflog can recover commits after accidental reset!
    Submodules

    Include other Git repositories within yours.

    git submodule add https://github.com/user/lib.git
    •
    Worktrees

    Work with multiple branches simultaneously.

    git worktree add ../feature-branch feature
    •
    Searching

    Search through code and commit history.

    git grep "TODO"
git log -S "function"
