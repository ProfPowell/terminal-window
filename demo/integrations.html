Framework Integrations

<terminal-window> is a vanilla web component that works with any JavaScript framework. Choose your framework below for installation instructions, code examples, and live demos.

React

Use refs and useEffect for event handling. Includes TypeScript types and Next.js support.

Guide Live Demo
Vue

Native web component support with isCustomElement config. Works with Vue 3 and Nuxt 3.

Guide Live Demo
Svelte

Works out of the box with excellent native support. Compatible with Svelte 4, 5, and SvelteKit.

Guide Live Demo
Astro

Perfect for static sites. No hydration needed for static content, optional client directives.

Guide Live Demo
Eleventy (11ty)

Static HTML generation with passthrough copy. Great for documentation and tutorial sites.

Guide Live Demo
Don't see your framework? Since terminal-window is a standard web component, it works with any framework that supports custom elements. Check the API Reference for the core usage pattern.
