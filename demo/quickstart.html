Quickstart

Get terminal-window running in under 2 minutes. See it in action below!

HTML Code
<!DOCTYPE html>
<html>
<head>
  <script type="module"
    src="https://unpkg.com/terminal-window">
  </script>
</head>
<body>
  <terminal-window
    title="Hello Terminal"
    enable-vfs="true"
    autofocus
  ></terminal-window>

  <script type="module">
    const terminal = document.querySelector('terminal-window');
    await customElements.whenDefined('terminal-window');

    // Create a greeting file
    terminal.fileSystem.writeFile('/greeting.txt',
      'Hopefully I am useful web component...');

    // Run hello world commands
    terminal.executeCommand(
      'echo "hello world I am a fun terminal simulator"'
    );
    terminal.executeCommand('cat /greeting.txt');
  </script>
</body>
</html>

Try It Yourself

  1. The terminal above is running the quickstart code
  2. Type ls to see files in the virtual file system
  3. Type cat /greeting.txt to read the greeting
  4. Type help to see all available commands
  5. Try mkdir mydir then cd mydir

Install via npm

npm install terminal-window
