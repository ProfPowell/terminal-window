Get Started

1. Include the Component

Add the script tag to your HTML page.

<script type="module"
  src="https://unpkg.com/terminal-window">
</script>
Or via npm: npm install terminal-window
2. Add the Element

Place the terminal-window element in your page.

<terminal-window
  theme="dark"
  prompt="$ "
  enable-vfs
  autofocus
></terminal-window>
