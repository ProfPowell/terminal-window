Apache Command Reference

Service Control

Start, stop, and manage the Apache service.

# Using systemctl (modern)
sudo systemctl start apache2

# Using apachectl
sudo apachectl start
    •
    Tip: Use reload instead of restart to apply config changes without dropping connections.
    apachectl Commands

    Apache control interface for managing the server.

    apachectl -t      # Test configuration
apachectl -S      # Show virtual hosts
    •
    Module Management

    Enable and disable Apache modules.

    sudo a2enmod rewrite
sudo a2dismod status
    •
    Note: After enabling/disabling modules, restart Apache: sudo systemctl restart apache2
    Virtual Host Management

    Enable and disable virtual host configurations.

    sudo a2ensite example.conf
sudo a2dissite 000-default.conf
    •
    Configuration Files

    Key Apache configuration files and directories.

    /etc/apache2/apache2.conf
/etc/apache2/sites-available/
/etc/apache2/mods-available/
    •
    Virtual Host Examples

    Common virtual host configuration patterns.

    <VirtualHost *:80>
    ServerName example.com
    DocumentRoot /var/www/example
</VirtualHost>
    •
    SSL/TLS Configuration

    Configure HTTPS and SSL certificates.

    sudo a2enmod ssl
sudo a2ensite default-ssl
    •
    Production: Use certificates from Let's Encrypt or a trusted CA for production sites.
    Log Files

    Apache access and error logs.

    tail -f /var/log/apache2/error.log
tail -f /var/log/apache2/access.log
    •
    .htaccess Examples

    Common .htaccess configurations.

    RewriteEngine On
RewriteCond %{HTTPS} off
RewriteRule ^(.*)$ https://%{HTTP_HOST}/$1 [R=301,L]
    •
    URL Rewriting

    mod_rewrite for clean URLs and redirects.

    RewriteRule ^product/([0-9]+)$ product.php?id=$1 [L]
    •
    Reverse Proxy

    Configure Apache as a reverse proxy.

    ProxyPass /api http://localhost:3000/
ProxyPassReverse /api http://localhost:3000/
    •
    Security

    Security hardening for Apache.

    ServerTokens Prod
ServerSignature Off
    •
    Tip: Always hide Apache version with ServerTokens Prod in production.
    Performance Tuning

    Optimize Apache for better performance.

    <IfModule mod_deflate.c>
  AddOutputFilterByType DEFLATE text/html
</IfModule>
